# sports_coach


A simple talking program to help me to do exercise. Fully voice controlled, nothing to read.

Main is talking coach.

Launch in English :
  ./talking_coach.py (Not yet tested in English.)

Launch in French :
  ./talking_coach.py fr
  
You can modify and save your preferences.
